# Global Store Power BI Dashboard

This repository contains the raw dataset used to create interactive dashboards in Microsoft Power BI.

## File Contents

- `Globalstore-Data.csv` — The source dataset

## Description

Power BI visualizations focus on:
- Sales by Region
- Profit Trends
- Product Category Performance
- Order Priority Analysis

## Tools Used

- Microsoft Power BI
- CSV Dataset
